## 🔑 Setting up the agent

After the installation, you must link your computer to your account:

1.  Open the DWService application if it did not start automatically.
2.  The application will ask for your DWService account credentials. If you do not have an account, visit the provider website to register.
3.  Enter your username and password.
4.  Assign a name to your computer so you can identify it later in your dashboard.
5.  Select "Register" to finish the process.

## 🌏 Controlling your computer remotely

Once the agent is active, you can access your machine from any device:

1.  Open a web browser on another computer, tablet, or phone.
2.  Go to the official DWService login page.
3.  Log in with the same account credentials used during the setup on your Windows machine.
4.  A list of your registered computers will appear. 
5.  Click on the name of your computer in the list.
6.  Select the "Remote Desktop" option from the menu.
7.  A web-based window will open showing your desktop. You can now use your mouse and keyboard to interact with the remote machine.

## 📁 File management features

The tool includes a file manager to move data between devices:

1.  Select the "Files" tab while viewing your computer in the browser.
2.  Browse your folders just like you do in Windows File Explorer.
3.  Click "Upload" to send files from your current device to the remote machine.
4.  Click "Download" to pull files from the remote machine to your local device.

## 🛡️ Security and privacy

DWService uses encrypted connections to keep your data safe. The connection between your computer and the server uses Transport Layer Security (TLS). Only you can see your screen or access your files when you log in to your account. Set a strong password for your account to prevent unauthorized access.

## ❓ Frequently asked questions

**Does the software run in the background?**
Yes, the agent runs as a system service. It stays active even if you log out of your Windows user account.

**Can I use this on multiple computers?**
Yes. You can install the software on every machine you want to manage. They will all appear in your main account dashboard.

**What happens if my computer goes to sleep?**
The remote connection requires the computer to be awake and connected to the internet. Adjust your Windows power settings to prevent the computer from entering sleep mode if you need constant remote access.

**Do I need a port forward on my router?**
No. The application handles the connection securely through the DWService servers. You do not need to change your firewall or router settings.

**How do I uninstall the software?**
Open the "Apps & features" menu in Windows settings. Search for DWService in the list, select it, and choose "Uninstall".
